Alerta Médico

Política de Privacidade

Última atualização: junho de 2025

🔐 Resumo — o essencial em 3 linhas

Seus dados de saúde ficam somente no seu celular. Não coletamos, não armazenamos e não compartilhamos nenhuma informação pessoal.

Esta política descreve como o aplicativo Alerta Médico trata suas informações.

📱 Dados armazenados localmente

O Alerta Médico armazena as seguintes informações exclusivamente no seu dispositivo, em um banco de dados local (SQLite) que nunca é sincronizado com servidores externos:

  • Nome completo
  • Tipo sanguíneo
  • Data de nascimento
  • Alergias
  • Lista de medicamentos (nome genérico, comercial, dose)
  • Contatos de emergência (nome, telefone, parentesco)
  • Lembretes de medicamentos e horários configurados
  • Observações médicas pessoais

Esses dados são classificados como dados sensíveis pela Lei Geral de Proteção de Dados (Lei 13.709/2018, art. 5º, II) por serem dados de saúde. O Alerta Médico respeita essa classificação ao não coletá-los ou transmiti-los.

🚫 O que NÃO fazemos

  • Não pedimos e-mail, senha ou qualquer cadastro
  • Não enviamos seus dados para servidores
  • Não vendemos ou compartilhamos informações
  • Não usamos rastreamento de uso ou analytics
  • Não exibimos anúncios
  • Não acessamos contatos, câmera, microfone ou localização

📡 Conexão à internet

O aplicativo usa conexão à internet apenas para uma finalidade técnica:

  • Atualização da base de medicamentos: periodicamente, o app verifica se há uma versão mais completa do banco de medicamentos e interações hospedado em CDN público. Nenhum dado pessoal é enviado nessa requisição — é um download apenas de leitura.

O app funciona completamente offline para todas as funcionalidades principais (tela de bloqueio, lembretes, interações com os dados já carregados).

🔔 Notificações e permissões

O Alerta Médico solicita as seguintes permissões do Android:

  • Notificações: para exibir a ficha médica na tela de bloqueio e enviar lembretes de medicamentos
  • Alarmes exatos: para disparar lembretes nos horários configurados com precisão
  • Inicialização automática: para restaurar a notificação de emergência após reinício do celular

Nenhuma dessas permissões é usada para rastrear ou coletar dados.

🏥 Tela de bloqueio

Quando o alerta de emergência está ativo, as informações do seu perfil médico (nome, tipo sanguíneo, medicamentos e alergias) ficam visíveis na notificação da tela de bloqueio do Android.

Isso é intencional e necessário para que socorristas possam acessar as informações sem precisar desbloquear o celular. Você pode desativar o alerta a qualquer momento na tela Início do aplicativo.

🌿 Recurso de reporte de medicamentos (opcional)

Se você pesquisar um medicamento que não está na nossa base de dados, o app oferece a opção de reportar o nome para que possamos adicioná-lo em atualizações futuras.

Ao usar esse recurso voluntário, apenas o nome do medicamento é enviado — de forma anônima, sem qualquer dado pessoal associado.

🗑️ Exclusão de dados

Como todos os dados estão no seu dispositivo, você pode excluí-los a qualquer momento:

  • Dados específicos: apague diretamente no app (medicamentos, contatos, perfil)
  • Todos os dados: desinstale o aplicativo — todos os dados são removidos automaticamente pelo Android

👶 Crianças

Este aplicativo não é direcionado a crianças menores de 13 anos e não coleta dados de menores de forma intencional.

⚠️ Aviso médico

O Alerta Médico é uma ferramenta de informação e organização pessoal. As informações sobre interações medicamentosas têm caráter exclusivamente educativo e não substituem avaliação médica ou farmacêutica profissional.

Consulte sempre seu médico antes de tomar decisões sobre seus medicamentos.
